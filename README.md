## 🤔 Xinsta?

- **[HOME: 유저들의 피드 보기]**  
  다른 사용자의 게시글을 확인하고 좋아요와 댓글을 달아보세요.
  
- **[SEARCH : 유저 검색하기]**  
  다른 사용자의 아이디를 검색하여 프로필을 확인하세요.

- **[ADD POST : 게시글 올리기]**<br>
  자신만의 사진을 멘트와 함께 게시글로 올려보세요.

- **[NOTIFICATION : 알림 확인하기]**  
  내 게시글에 달린 좋아요와 댓글을 확인하세요.

- **[PROFILE : 나의 프로필 확인하기]**  
  내 게시글과 회원정보를 확인하고 수정하세요.
